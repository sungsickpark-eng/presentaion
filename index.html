운영 방식

이 페이지는 카드 목록만 관리합니다. 실제 발표는 기존처럼 각각 독립된 HTML 파일로 두고, 아래 데이터 목록에 새 파일 정보만 추가하면 카드가 자동으로 렌더링됩니다.

새 자료 추가 순서

  1. 새 발표자료 HTML 파일을 같은 폴더에 추가합니다.
  2. 이 페이지 하단 스크립트의 presentations 배열에 항목 하나를 추가합니다.
  3. 제목, 설명, 태그, 파일명만 입력하면 카드가 자동 생성됩니다.

발표자료 라이브러리

주제별로 걸러보거나 최신 순으로 빠르게 탐색할 수 있게 구성했습니다.

선택한 태그에 해당하는 발표자료가 없습니다.

카드 순서는 최신 번호가 먼저 오도록 자동 정렬됩니다.
